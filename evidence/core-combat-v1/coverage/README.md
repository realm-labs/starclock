# Goal 01 coverage evidence

`goal-coverage.json` is the initial completeness report for the frozen Goal 01
manifests. `coverage-index.json` binds it by SHA-256.

The report includes the Standard profile in the total completeness denominator,
so the complete initial accounting is 283 required entries: 88 released combat
forms, 165 Light Cones, 17 enemy variants, six encounters, six scenarios and one
profile. Rin Tohsaka and Gilgamesh are disabled audit entries outside that
denominator.

Initial `DataReady` is correctly zero. The prepared JSON pack is evidence only,
and no production Excel/Sora catalog exists yet.
