# Character partition C02 production import

`G01-P7-C02` promotes the second frozen character partition through the
repository-owned Excel/Sora production path. Its membership is Black Swan,
Blade, Boothill, Bronya, Castorice, Cerydra, Cipher and Cyrene.

The deterministic author reads only the prepared Version 4.4 pack with digest
`0dca8ae581b4fa1e9fe8ce0c9e67ac6eb72c251deacbd4831751ce685e45ef5a`.
It adds 8 character definitions, 53 abilities with 1,807 level-parameter
values, 151 battle-relevant Trace nodes and complete E1-E6 sets (48 Eidolons).
The import preserves each prepared record's evidence and mechanism quality.

The partition owns internal IDs `40000..49999`. Partition rows replace their
owned workbook slice in place, so V1B, C01 and C02 deterministic checks remain
composable after later partitions land. Schema-level enhanced Basics remain in
their Basic combat family while retaining explicit no-SP-generation semantics.
HP-scaling damage bindings are authored for Blade and Castorice, and Blast/AoE
overrides retain the prepared target geometry.

Castorice has no conventional Energy cap. Her form instead defines the
`newbud` resource with a normalized maximum of 100, and `Doomshriek, Dawn's
Chime` spends exactly 100 at `ActionStarted`. The generic M13 boundary lowers
that row without synthesizing Energy, uses it for Ultimate legality, and pays it
inside the rollback transaction. C02 goldens also pin Blade's two-component
HP-scaling enhanced Basic and compile every production form at E0 and complete
E6.
