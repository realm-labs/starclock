# Goal 01 Frozen Content Manifests

These generated manifests are the completeness oracle for `core-combat-v1`.
They bind the prepared Version 4.4 content-reference pack, freeze all 88 released
combat forms and 165 released Light Cones, select the representative public
`standard-v1` subset, and assign stable Phase 7 partitions.

The JSON files are generated by `tools/goal-manifest/generate.mjs`; do not edit
them by hand. Their aggregate digest is recorded in `manifest-index.json` and in
the Goal 01 status ledger.

`Required` is an inclusion decision. `Pending` records that the referenced fact
has not yet crossed the Excel/Sora/Rule-IR `DataReady` boundary. The prepared
JSON remains evidence and deterministic bootstrap input only; it is not a
runtime catalog.
